## 简介

这是一个使用pyxel引擎开发的飞机大战游戏

学习体验一款游戏完整的开发流程

像素作画、动画效果、键盘控制、敌人飞行AI、碰撞检测、BGM制作等


### 安装运行
模块安装
pip install pyxel



### 编辑器工具
pyxeleditor ircraftWars.pyxel
