# 一个Python飞机大战像素游戏

## 简介

这是一个使用pyxel引擎开发的飞机大战游戏
体验完整的游戏开发流程
流程组要是像素作画、动画效果、键盘控制、敌人飞行AI、碰撞检测、BGM制作等


### 安装运行
模块安装
pip install pyxel



### 编辑器工具
pyxeleditor ircraftWars.pyxel
